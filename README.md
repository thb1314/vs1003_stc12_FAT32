# 基于VS1003的MP3播放器
##硬件

*	主控MCU:STC12LE5A60S2(3.3V 52增强型单片机)
	*	自带硬件SPI
	*	1024 Byte 外部RAM和 256 Byte 内部RAM
	*	64K Flash存储
*	VS1003B MP3解码芯片

##软件

*	FAT32文件系统在嵌入式系统的部分移植

##实现功能
*	自动搜索根目录下面的全部以MP3结果的文件。可以自动顺序播放
*	可以自动顺序播放，也可以通过外部中断0（下降沿触发）实现下一曲的切换
*	外部中断1（下降沿触发）实现音量的增减
